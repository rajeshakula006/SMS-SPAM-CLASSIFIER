# SMS Spam Classifier (Mini Project)
A simple Machine Learning project built for B.Tech 2nd Year (AI & Machine Learning).

## Overview
This project uses the **Naive Bayes Algorithm** to classify text messages as either 'Spam' or 'Ham' (Not Spam). It demonstrates text vectorization and supervised learning.

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, Scikit-learn
- **Algorithm:** Multinomial Naive Bayes

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python sms_classifier.py`
